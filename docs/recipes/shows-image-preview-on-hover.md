# 画像のホバー時プレビューの表示

この #レシピ を使用すると、画像にホバーしたときにプレビューを表示できます。

拡張機能: [Image preview](https://marketplace.visualstudio.com/items?itemName=kisstkondoros.vscode-gutter-preview) を使用すると、ガターとホバー時に画像プレビューが表示されます。
